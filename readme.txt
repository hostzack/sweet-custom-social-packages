Use WP to manage social packages like facebook likes, facebook followers, youtube likes, youtubes viewrs etc. It supports paypal.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the options in the Tools->Sweet Custom Social Packages admin page
4. Publish your packages on your blog (best in the sidebar) with the shortcodes [Social_Package id=''].
